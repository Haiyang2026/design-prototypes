#### 1.2 字段联动关系

##### 1.2.1 领用类型 → 需求明细子表

**关联逻辑**：
- 需先选择【领用类型】，才可填写需求明细子表内容

##### 1.2.2 实际领用人 → 岗位、组织全名、职等薪级、性别、入职时间

**关联逻辑**：
- 选择实际领用人后，系统自动获取并填充以下字段：岗位、组织全名、职等薪级、性别、入职时间
- 以上字段均为只读，不可手动修改

##### 1.2.3 实际领用人 → 归属成本中心

**关联逻辑**：
- 系统根据实际领用人自动带出归属成本中心
- 不可手动修改

##### 1.2.4 实际领用人 → 工作服（可选范围）

**关联逻辑**：
- 选择工作服时弹出选框，展示数据来源于【工作服价格配置表】
- 系统自动匹配【领用标准配置表】，根据实际领用人的组织、职等、是否为操作组等维度，筛选出当前人员符合领取规范的服装选项
- 不符合领用标准的服装不在选框中展示

##### 1.2.5 工作服 → 单价

**关联逻辑**：
- 选择工作服后，系统自动从【工作服价格配置表】带出对应系统单价，用于后续成本计算

##### 1.2.6 申请人组织 → 领用仓库

**关联逻辑**：
- 领用仓库默认匹配申请人所在组织对应的仓库（从【仓管员配置表】获取）
- 用户可手动修改为其他仓库

### 2. 流程

#### 2.1 审批步骤总览

| 步骤ID | 步骤名称 | 环节属性 | 审批人获取方式 | 审批方式 | 是否任务环节 | 是否允许退回 | 相同审批人自动跳过 | 为空跳过 | 说明 |
|--------|----------|----------|----------------|----------|--------------|--------------|--------------------|----------|------|
| step 1 | 提单 | 执行 | 表单用户（当前登录用户） | 单审 | 否 | - | 是 | 否 | 员工填写并提交领用申请 |
| step 2 | 仓管员确认 | 执行 | 业务字段（领用仓库对应仓管员，取自【仓管员配置表】） | 单审 | 是 | 是 | 是 | 否 | 仓管员确认领用申请 |
| step 3 | 超领判断 | 机器自动 | 系统自动 | 机器自动 | 否 | 否 | - | - | 系统自动判断是否超领，无需人工干预 |
| step 4 | 超领确认 | 审批 | 业务字段（申请人本人） | 单审 | 是 | 是 | 是 | 否 | 仅超领时触发，生成超领确认单给申请人确认 |
| step 5 | 流程结束 | - | - | - | - | - | - | - | 无超领时告知申请人确认后结束；超领确认后结束 |

#### 2.1.1 审批流程图

![工作服领用申请审批流程图](工作服领用申请_审批流程图_V1.0.svg)

**流程说明**：
1. 提单提交后，申请单流转到对应仓库的仓管员处
2. 仓管员确认后，系统自动判断本次领用是否超领（对比领用标准配置表）
3. **超领情况**：生成超领确认单给申请人本人确认
4. **无超领情况**：告知申请人无超领，确认是否继续领用后进入下一流程
5. 超领判断逻辑由系统自动完成，不需要人工干预

### 3. 配置表

#### 3.1 仓管员配置表

**目的**：按组织维度配置领用仓库与对应仓管员的映射关系。

| 配置项 | 说明 | 配置方式 |
|--------|------|----------|
| 组织 | 组织维度 | 选择对应组织 |
| 领用仓库 | 该组织对应的领用仓库 | 录入仓库信息 |
| 仓管员 | 该仓库对应的仓管员 | 配置仓管员人员 |

**配置逻辑**：
- 先选择对应组织 → 录入该组织对应的领用仓库 → 配置该仓库对应的仓管员
- 所有配置信息统一存储在该配置表中
- 提单时，系统根据申请人所在组织自动匹配对应仓库和仓管员

#### 3.2 工作服尺码配置表

**目的**：按工作服名称单独设置对应尺码及身高围度参数。

| 尺码 | 身高围度参数 |
|------|-------------|
| XXS | 150/76A |
| XS | 160/80A |
| S | 165/84A |
| M | 170/88A |
| L | 175/92A |
| XL | 180/96A |
| XXL | 185/100A |
| XXXL | 190/104A |
| XXXXL | 195/108A |
| 5XL | 200/110A |

**配置逻辑**：
- 按工作服名称单独设置对应尺码（每个工作服名称绑定一套尺码配置）
- 共10个尺码档位（XXS ~ 5XL）

#### 3.3 工作服价格配置表

**目的**：基于尺码配置表搭建，定义工作服的类别、季节、品类、适用性别、尺码范围和系统单价。

| 序号 | 字段 | 说明 | 可选值/备注 |
|------|------|------|------------|
| 1 | 类别 | 工作服类别 | 厂服、形象服 |
| 2 | 季节 | 适用季节 | 冬季、夏季 |
| 3 | 工作服简称 | 工作服名称 | 文本录入 |
| 4 | 服装品类 | 服装品类分类 | 1短袖衬衫、2长袖衬衫、3西裤、4西裙、5西装外套、6短袖上衣、7外套、8裤子 |
| 5 | 适用性别 | 适用性别 | 男、女 |
| 6 | 尺码类型 | 尺码分类 | 标准码、定制码 |
| 7 | 尺码范围 | 适用职等范围 | 六职等以下、六职等以上 |
| 8 | 系统单价 | 工作服单价 | 文本录入（数值） |
| 9 | 物料编码 | 物料编号 | 文本录入 |

#### 3.4 领用标准配置表

**目的**：定义员工在不同组织、职等条件下的领用标准和额度，系统提单时自动匹配筛选符合规范的服装。

| 配置维度 | 说明 | 可选值 |
|----------|------|--------|
| 组织维度 | 适用的组织范围 | 选择组织 |
| 是否操作组 | 区分是否为操作组 | 是、否 |
| 职等 | 按职等划分 | 四职等以上、四职等以下 |

| 配置项 | 说明 | 可选值/备注 |
|--------|------|------------|
| 类别 | 工作服类别 | 厂服、行销服 |
| 季节 | 服装季节 | 复用工作服价格配置表数据 |
| 服装品类 | 服装品类 | 复用工作服价格配置表基础数据 |
| 领用年限 | 领用周期 | 入职第一年、入职第二年 |
| 领用额度 | 可领用数量 | 纯数字设置，如1套或2套 |

**匹配逻辑**：
- 员工提单选择工作服时，系统自动匹配该配置表
- 根据实际领用人的组织、职等、入职时间（判断第几年）等维度筛选
- 仅展示符合领取规范的服装选项
- 自动带出对应工作服的系统单价

#### 3.5 成本中心统计（系统输出）

**目的**：系统根据申请单内容自动生成成本中心汇总统计表，无需手动汇总。
