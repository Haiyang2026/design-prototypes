# 平台自存数据需求

> 版本：v0.1 | 日期：2026-07-14 | 状态：草稿，待确认后回填需求文档  
> 说明：本文档记录我方平台需要自行存储的用户个性化数据，不涉及外部接口

---

## 1. 应用中心 — 用户自存数据

### 1.1 置顶顺序

| 字段 | 说明 |
|------|------|
| 用户ID | — |
| 置顶应用ID列表 | 数组，按用户拖拽后的顺序排列 |
| 管理员默认置顶列表 | 管理员预设的初始化置顶，用于新用户初始化 + 恢复默认 |

**规则：**
- 置顶区按此列表顺序展示
- 用户拖拽后更新顺序
- 取消置顶 = 从列表中移除该应用ID
- 新用户初始化：加载管理员默认置顶列表
- 恢复默认：覆盖为管理员默认置顶列表

### 1.2 收藏列表

| 字段 | 说明 |
|------|------|
| 用户ID | — |
| 收藏应用ID列表 | 数组，用户收藏的应用ID |

**规则：**
- 收藏仅用于 ⭐ 标记 + 管理弹窗「我的收藏」tab 展示
- 不参与排序，不影响卡片展示顺序
- 收藏与置顶完全独立，互不影响
- 取消收藏 = 从列表中移除该应用ID

## 2. 顶部菜单 — 用户自存数据

### 2.1 菜单顺序

| 字段 | 说明 |
|------|------|
| 用户ID | — |
| 菜单项顺序列表 | 数组，按用户拖拽后的顺序排列的菜单项ID |

**规则：**
- 初始顺序 = 系统默认（或管理员配置）
- 用户在页面设置中拖拽后更新顺序
- 恢复默认 = 覆盖为系统默认顺序

### 2.2 菜单收藏列表

| 字段 | 说明 |
|------|------|
| 用户ID | — |
| 收藏菜单项ID列表 | 数组，用户收藏的菜单项ID |

**规则：**
- 收藏仅用于快速筛选和个人标记
- 不参与排序
- 与应用中心收藏逻辑一致

### 2.3 默认门户页

| 字段 | 说明 |
|------|------|
| 用户ID | — |
| 默认门户页ID | 用户设置的首选门户页 |

**规则：**
- 默认全员 = 集团门户
- 管理员可在后台为特定用户/角色配置其他默认页
- 后台有配置时，对应账号优先展示其他默认页
- 用户自己也可在设置中修改

---

## 3. 数据删除 / 同步规则

| 场景 | 行为 |
|------|------|
| 上游数据源删除某应用/菜单项 | 接口不再返回 → 前端拉不到 → 不在列表中展示。自存数据中若有该ID，也同步移除 |
| 用户切换设备 | 自存数据跟随用户ID，跨设备同步 |
| 管理员修改默认置顶/默认菜单 | 对未操作过的用户自动更新；已操作过的用户遵循个人自定义优先 |
